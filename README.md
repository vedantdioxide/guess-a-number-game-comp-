# guess-a-number-game-
USER:
Guess the Number is a fun, interactive command-line game where the computer picks a random number between 1 and X, and you try to guess it with limited attempts. Built as a beginner project to practice logic, loops, and user input in Python.
COMP:
 Interactive CLI game with two modes—player guesses computer's secret number (1-100) or computer guesses player's number using binary search logic. Features hints (higher/lower), and win tracking. Beginner Python project to master loops, conditionals, random module, and user input."
